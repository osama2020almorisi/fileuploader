# 🛍️ نظام إدارة المتجر الإلكتروني

نظام متكامل لإدارة المتاجر الإلكترونية مع واجهة إدارة قوية وعرض منتجات تفاعلي وسلة تسوق ذكية.

![صورة النظام](https://via.placeholder.com/800x400?text=نظام+إدارة+المتجر+الإلكتروني)

## ✨ المميزات الرئيسية

### 🎯 لوحة التحكم الإدارية
- **إدارة المنتجات**: إضافة وتعديل وحذف المنتجات بسهولة
- **التصنيفات**: تنظيم المنتجات في فئات متعددة
- **رفع الصور**: دعم رفع متعدد للصور مع معاينة فورية
- **التكامل مع GitHub**: تخزين آمن للبيانات على منصة GitHub

### 🛒 متجر أمامي متكامل
- **عرض المنتجات**: واجهة عرض منتجات جذابة وتفاعلية
- **سلة تسوق ذكية**: إدارة كاملة لعملية الشراء
- **خيارات دفع متعددة**: دعم وسائل دفع متنوعة
- **تجربة مستخدم محسنة**: تصميم متجاوب يعمل على جميع الأجهزة

### 📱 واجهة مستخدم متقدمة
- **تصميم عصري**: واجهة مستخدم عصرية وسهلة الاستخدام
- **تفاعلية عالية**: تأثيرات وتحسينات للأداء
- **دعم اللغة العربية**: تصميم كامل يدعم اللغة العربية
- **تجربة مستخدم متميزة**: تركيز على سهولة الاستخدام والكفاءة

4. اختر صور المنتج (اسحب وأسقط أو انقر للاختيار)
5. انقر على "حفظ المنتج"

## 🌟 مميزات فريدة

2. **واجهة السحب والإفلات**: رفع الصور بطريقة سهلة بالسحب والإفلات

6. **خيارات استلام متعددة**: واتساب، إيميل، رسائل نصية
